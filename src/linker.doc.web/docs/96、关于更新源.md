# 96、关于更新源

:::danger[官方]

官方更新源可能不稳定，建议自定义

:::

:::tip[自定义]

1. 存放格式，按默认官方的配置举例 https://static.snltty.com/downloads/linker
    1. 版本文件：https://static.snltty.com/downloads/linker/version.txt
    2. 程序文件：https://static.snltty.com/downloads/linker/版本号/压缩包.zip
    3. 示例：https://static.snltty.com/downloads/linker/v1.9.99/linker-linux-x64.zip

2. 配置方式，可以做个定时任务从GitHub 下载配置文件，并放在程序目录下
    1. 版本文件：https://raw.githubusercontent.com/snltty/linker/refs/heads/master/version.txt
    2. 程序文件：https://github.com/snltty/linker/releases/download/v1.9.99/linker-linux-x64.zip

:::
