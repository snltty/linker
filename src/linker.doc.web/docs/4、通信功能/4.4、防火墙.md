# 4.4、防火墙

:::tip[说明]
1. 防火墙影响 虚拟网卡、端口转发、socks5代理
2. 开启防火墙后默认阻止所有连接，按顺序从上到下匹配，匹配成功后停止
3. `原设备` * 表示所有
4. `目标IP` *、0、0.0.0.0/0 表示所有，掩码缺省值32
5. `目标端口` *、0 表示所有、80单个、80-88范围、80,443多个
