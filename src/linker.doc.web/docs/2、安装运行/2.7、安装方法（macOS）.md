#  2.7、安装方法（macOS）

:::tip[说明]

linker-osx-x64 或 linker-osx-arm64

```
cd linker-osx-arm64

//清除隔离标记
xattr -dr com.apple.quarantine linker-osx-arm64

//本地签名
codesign --force --deep --sign - ./linker
//可选的验证签名
//codesign -dv --verbose=4 ./linker

chmod +x linker
sudo ./linker
```
:::
